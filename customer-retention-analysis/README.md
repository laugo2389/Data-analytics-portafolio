# Customer Retention Analysis

## Objective
Analyze user retention and churn behavior to identify improvement opportunities.

## Tools
- SQL
- Python (Pandas)
- Google Sheets

## Dataset
Simulated e-commerce user activity dataset.

## Key Metrics
- Retention rate
- Churn rate
- Cohort analysis

## Results
Identified key drop-off points in the user lifecycle and proposed actions to improve retention.
