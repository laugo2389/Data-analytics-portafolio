# 📊 Andes Retail Dashboard

## Project Overview

This project presents an interactive Power BI dashboard developed to analyze retail sales performance across multiple countries, product categories, and time periods.

The objective was to transform raw business data into meaningful insights that support strategic decision-making through interactive visualizations and KPI tracking.

---

## Business Problem

Retail managers need a quick and reliable way to monitor business performance, identify profitable products, and compare sales across regions.

This dashboard was designed to answer those business questions efficiently.

---

## Objectives

- Analyze sales performance.
- Monitor revenue, profit, and margin.
- Compare countries and product categories.
- Track KPIs over time.
- Support data-driven decision making.

---

## Tools Used

- Power BI
- Power Query
- DAX
- Data Modeling
- Business Intelligence
- Data Visualization

---

## Dashboard Features

- KPI Cards
- Sales Trends
- Profit Analysis
- Category Performance
- Regional Comparison
- Interactive Filters

## Key Insights

- Identified the best-performing product categories.
- Compared sales across multiple countries.
- Measured profitability through KPI tracking.
- Built an interactive dashboard for business users.
