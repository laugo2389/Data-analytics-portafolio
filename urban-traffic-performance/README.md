# Urban Traffic Performance Analysis

## Business Problem
Analyze traffic congestion patterns to support data-driven urban mobility decisions.

## Dataset
Traffic metrics dataset including congestion, delays, and travel times across cities.

## Objectives
- Clean and prepare raw traffic data
- Perform exploratory data analysis
- Identify trends and high-impact congestion drivers
- Visualize insights for decision-making

## Tools
Python (Pandas), Data Cleaning, EDA, Data Visualization

## Outcome
Delivered actionable insights on congestion patterns and performance differences across cities.
