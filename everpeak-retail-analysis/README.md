# 🛍️ EverPeak Retail Analysis

## Project Overview

This project focuses on cleaning and preparing retail sales data for analysis using Python and Pandas.

The workflow includes handling missing values, detecting outliers, standardizing data, and preparing datasets for business analysis.

- Jupyter Notebook

- Data Preprocessing

- Outlier Detection

- Exploratory Data Analysis

- Standardized inconsistent records.
- Detected anomalous observations.
- Prepared clean data for business intelligence.
