# 🎯 Final Capstone Project

## Project Overview

This capstone project integrates the complete data analytics workflow, from data preparation to business insights and dashboard development.

Using SQL, Python, and Power BI, the project demonstrates the ability to clean, transform, analyze, and visualize data to support strategic business decisions.

---

## Business Problem

Organizations generate large amounts of data, but without proper analysis it is difficult to identify trends, measure performance, and make informed decisions.

The goal of this project was to transform raw data into meaningful business insights through a complete end-to-end analytics process.

---

## Objectives

- Clean and prepare the dataset.
- Perform exploratory data analysis (EDA).
- Validate business hypotheses using statistical analysis.
- Build interactive dashboards.
- Present actionable business recommendations.
