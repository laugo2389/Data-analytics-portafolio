# 🏠 Real Estate Sales Dashboard

## Project Overview

This project presents an interactive Power BI dashboard designed to analyze real estate sales performance.

The dashboard provides insights into revenue, profitability, customer behavior, and sales trends through dynamic visualizations and KPI monitoring.
