# 📊 ConnectaTel Statistical Analysis

## Project Overview

This project analyzes customer behavior using statistical hypothesis testing to support data-driven business decisions.

Python and SciPy were used to evaluate significant differences between customer groups and validate business hypotheses.

---

## Tools

- Python
- Pandas
- SciPy
- Matplotlib
- Jupyter Notebook

---

## Skills Demonstrated

- Statistical Analysis
- Hypothesis Testing
- A/B Testing
- Data Visualization
- Business Decision Support

---

## Key Insights

- Tested business hypotheses using statistical methods.
- Identified significant behavioral differences between customer segments.
- Supported decision-making with statistical evidence.
- Presented results through clear visualizations.
